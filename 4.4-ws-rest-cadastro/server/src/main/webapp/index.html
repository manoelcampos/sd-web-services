Servidor REST utilizando API REST-RS

O endereço base para acesso aos serviços REST é http://localhost:8080/server/api/. Este caminho é definido no arquivo RestConfig. Ele é um endereço incompleto, logo, precisa ser complementado como indicado abaixo.

A partir de tal endereço, cada classe anotada com @Path irá definir caminhos adicionais. Por exemplo, para acessar a API de acesso a dados de usuários, temos algumas URIs como:

  • http://localhost:8080/server/api/usuario/1 para buscar usuário pelo ID
  • http://localhost:8080/server/api/usuario/cpf/11111111111 para buscar usuário pelo CPF
