Servidor REST com framework Quarkus: REST-RS, CDI, JPA com Hibernate e Panache

O endereço base para acesso aos serviços REST é http://localhost:8080/api/. Este caminho é definido no arquivo RestConfig. Ele é um endereço incompleto, logo, precisa ser complementado como indicado abaixo.

A partir de tal endereço, cada classe anotada com @Path irá definir caminhos adicionais. Por exemplo, para acessar a API de acesso a dados de usuários, temos algumas URIs como:

  • /api/hello
  • /api/hello/{digite seu nome na URL}
  • /api/usuario/1 para buscar usuário pelo ID
  • /api/usuario/cpf/11111111111 para buscar usuário pelo CPF
  • /api/cliente/1 para buscar cliente pelo ID
